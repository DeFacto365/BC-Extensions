# Demos

Create one folder per demo or prospect.

Folder naming:

- Use the prospect/company name, for example `Panthera`.
- Keep all demo-specific assets in that folder.

Each demo folder can contain:

- `current-prospect.json`
- item request JSON
- sales order request JSON
- downloaded logos
- product pictures
- website scrape notes
- any other files tied only to that demo

Shared automation code stays in `../bc-repeatable/`.
Shared standards stay in `../main/`.
