# Demo Template

Copy this folder pattern for a new demo.

Suggested structure:

- `assets/` for logos, product pictures, and website images.
- `requests/` for item, attribute, company information, and sales order payloads.
- `current-prospect.json` for prospect name, prospect URL, BC company name, and source URLs.

Keep demo files out of the repository root and out of `bc-repeatable/`.
